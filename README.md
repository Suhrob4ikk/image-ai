# Image AI — управление изображением через нейронную сеть

Учебный проект по курсу компьютерного зрения. Программа принимает текстовую команду
на естественном языке, отправляет её в локальную языковую модель, которая определяет
нужную операцию и возвращает структурированный ответ. На основе этого ответа
выполняется соответствующая функция OpenCV.

## Принцип работы

Пользователь вводит команду в свободной форме — например, «поверни на 90 градусов»
или «сделай чёрно-белым». Команда передаётся языковой модели через Ollama. Модель
возвращает JSON-объект с указанием действия и параметров. Программа разбирает этот
ответ и вызывает соответствующую функцию OpenCV. Результат отображается рядом
с оригиналом через Matplotlib.

```
Команда пользователя
        ↓
Qwen 2.5 3B (локально через Ollama)
        ↓
{"action": "rotate", "angle": 90}
        ↓
cv2.warpAffine(image, M, (cols, rows))
        ↓
Вывод: оригинал + результат
```

## Поддерживаемые операции

| Команда | OpenCV функция |
|---|---|
| поверни на N градусов | `cv2.getRotationMatrix2D` + `cv2.warpAffine` |
| измени размер до WxH | `cv2.resize` |
| выдели красный / зелёный / синий канал | `cv2.split` |
| сделай чёрно-белым | `cv2.cvtColor(img, COLOR_BGR2GRAY)` |
| найди границы | `cv2.Canny` |
| отрази горизонтально / вертикально | `cv2.flip` |
| размой изображение | `cv2.GaussianBlur` |

Нейросеть понимает произвольные формулировки — «убери цвета», «переверни вверх ногами»,
«сделай размытым» — и сопоставляет их с доступными операциями.

## Стек

- Python 3.13
- OpenCV 4.13
- Ollama 0.24 — локальный сервер для запуска LLM
- Qwen 2.5 3B — языковая модель (работает без интернета)
- Matplotlib 3.10
- Tkinter — стандартный диалог выбора файла

## Установка зависимостей

```bash
pip install opencv-python matplotlib ollama
```

Также необходимо установить [Ollama](https://ollama.com) и загрузить модель:

```bash
ollama pull qwen2.5:3b
```

## Запуск

```bash
# 1. Запустить Ollama в отдельном терминале
ollama serve

# 2. Запустить программу
python image_ai.py
```

При запуске откроется окно выбора файла. После загрузки изображения можно вводить
команды в консоли. Для выхода — `выход` или `exit`.
