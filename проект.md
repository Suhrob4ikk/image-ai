# Проект: Управление изображением через нейронную сеть

## Что делает программа

Программа принимает текстовую команду на естественном языке (русском или английском),
отправляет её в локальную нейросеть, которая распознаёт намерение пользователя
и возвращает машинную инструкцию. Программа выполняет соответствующую операцию
над изображением через OpenCV и показывает результат рядом с оригиналом.

---

## Как это работает

```
Пользователь вводит текст
        ↓
Нейросеть (Llama / Qwen) читает команду
        ↓
Возвращает JSON: {"action": "rotate", "angle": 90}
        ↓
Программа вызывает нужную функцию OpenCV
        ↓
Matplotlib показывает оригинал и результат рядом
```

---

## Технологии и библиотеки

| Библиотека | Назначение |
|---|---|
| `cv2` (OpenCV) | Обработка изображений: поворот, фильтры, каналы, границы |
| `ollama` | Подключение к локальной нейросети через Ollama |
| `json` | Разбор JSON-ответа от нейросети |
| `matplotlib` | Отображение картинок: оригинал и результат рядом |
| `tkinter` | Окно выбора файла (стандартная библиотека Python) |

**Нейросеть:** модель `qwen2.5:3b` (3 миллиарда параметров), работает **локально** через Ollama — без интернета и без платных API.

## Поддерживаемые команды (7 операций)

| Команда пользователя | Действие |
|---|---|
| `поверни на 90` / `на 45` / `на 130` | Поворот на произвольный угол |
| `измени размер до 400x300` | Изменение разрешения |
| `выдели красный / зелёный / синий канал` | Извлечение цветового канала |
| `сделай чёрно-белым` | Перевод в оттенки серого |
| `найди границы` | Детекция краёв (алгоритм Кэнни) |
| `отрази горизонтально / вертикально` | Зеркальное отражение |
| `размой изображение` | Размытие по Гауссу |

## Ключевые методы OpenCV

### Поворот на произвольный угол
```python
rows, cols = image.shape[:2]
M = cv2.getRotationMatrix2D((cols/2, rows/2), -angle, 1)
result = cv2.warpAffine(image, M, (cols, rows))
```
`getRotationMatrix2D` строит матрицу аффинного преобразования для поворота вокруг центра.  
`warpAffine` применяет эту матрицу к каждому пикселю изображения.

### Изменение размера
```python
result = cv2.resize(image, (width, height))
```
Масштабирует изображение до указанных пикселей интерполяцией.

### Выделение цветового канала
```python
B, G, R = cv2.split(image)
```
OpenCV хранит пиксели в порядке BGR. `split` разбивает картинку на три одноканальных матрицы.

### Перевод в оттенки серого
```python
result = cv2.cvtColor(image, cv2.COLOR_BGR2GRAY)
```
Смешивает три канала в один по формуле яркости.

### Детекция краёв (Canny)
```python
gray  = cv2.cvtColor(image, cv2.COLOR_BGR2GRAY)
result = cv2.Canny(gray, 100, 200)
```
Алгоритм Кэнни находит резкие перепады яркости — это и есть границы объектов.  
100 и 200 — нижний и верхний пороги чувствительности.

### Зеркальное отражение
```python
result = cv2.flip(image, 1)   # 1 — горизонтально, 0 — вертикально
```

### Размытие по Гауссу
```python
result = cv2.GaussianBlur(image, (15, 15), 0)
```
Каждый пиксель заменяется средним взвешенным соседей в области 15×15.

## Как нейросеть понимает команду

Программа формирует **промпт** — текстовую инструкцию для модели:

```
Ты — помощник по обработке изображений.
Определи действие и верни ТОЛЬКО JSON.

Доступные действия:
- {"action": "rotate", "angle": 90}
- {"action": "grayscale"}
...

Команда пользователя: "сделай чёрно-белым"
```

Модель возвращает: `{"action": "grayscale"}`  
Программа читает поле `action` и вызывает соответствующую функцию OpenCV.

## Как запустить

1. Запустить Ollama: `ollama serve`  
2. Запустить программу: `python image_ai.py`  
3. Выбрать картинку в открывшемся окне  
4. Вводить команды на русском или английском
